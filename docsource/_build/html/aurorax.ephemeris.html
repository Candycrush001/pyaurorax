aurorax.ephemeris.get_ephemeris_sources(program=None, platform=None, instrument_type=None, source_type=None, owner=None, format='basic_info')¶

Returns a list of dictionaries representing all ephemeris sources.

Parameters

  • program – program name to filter sources by, optional

  • platform – platform name to filter sources by, optional

  • instrument_type – instrument type to filter sources by, optional

  • source_type – source type to filter sources by (e.g. “heo”), optional

  • owner – owner ID to filter sources by, optional

  • format – the format of the ephemeris source returned Available values: “identifier_only”, “basic_info”, “full_record”. Defaults to “basic_info”.
