class aurorax.conjunctions.ConjunctionEvent(conjunction_type=None, e1_source=None, e2_source=None, start=None, end=None, min_distance=None, max_distance=None)¶

Bases: object

Represents a conjunction event between two instruments.

Variables

  • conjunction_type – [“nbtrace”], [“sbtrace”], or [“nbtrace”, “sbtrace”]

  • e1_source – dictionary representation of the first ephemeris source in the conjunction

  • e2_source – dictionary representation of the second ephemeris source in the conjunction

  • start – datetime object representing the start of the conjunction

  • end – datetime object representing the end of the conjunction

  • min_distance – the minimum distance in kilometers between ephemeris sources in this conjunction event

  • max_distance – the maximum distance in kilometers between ephemeris sources in this conjunction event

a dictionary representation of the conjunction event

class aurorax.conjunctions.ConjunctionSearch¶

Bases: object

A class representing the parameters required for a single conjunction search.

Variables

  • ground_instrument – dictionary representation of the ground instrument in this conjunction search

  • space_instrument – dictionary representation of the space instrument in this conjunction search

  • start – datetime object representing the start of the search period

  • end – datetime object representing the end of the search period

  • conjunction_types – one of [“nbtrace”], [“sbtrace”], or [“nbtrace”, “sbtrace”]

  • max_distance – the maximum distance between instruments for a conjunction. Defaults to 500 km.

add_ground(programs=None, platforms=None, instrument_types=None, metadata_filters=None)¶

Adds parameters for a ground instrument to the conjunction search.

Parameters

  • programs – a list of strings representing the ground instrument programs to search

  • platforms – a list of strings representing the ground instrument platforms to search

  • instrument_types – a list of strings representing the ground instrument instrument types to search

  • metadata_filters – a list of dictionaries representing metadata filters for the ground instrument in the form {“key”: <key>, “operator”: <operator>, “value”: <value>}. E.g. {“key”: “temperature”, “operator”: “=”, “value”: 25.0}

set_all_max_distances(distance=500)¶

Sets the maximum distance between all pairs of instruments whose maximum distance has not been explicitly set using set_max_distance. This method must be called before execution of the conjunction search.

Parameters

distance – a positive numeric value representing the maximum distance between instruments

set_max_distance(instrument1, instrument2, distance)¶

Sets the maximum distance between two instruments in this multi-conjunction search.

Parameters

  • instrument1 – “ground<list_number>” or “space<list_number>” representing the space/ground isntrument’s position in the ground_instruments or space_instruments list, e.g. “ground1”

  • instrument2 – “ground<list_number>” or “space<list_number>” representing the space/ground isntrument’s position in the ground_instruments or space_instruments list, e.g. “space1”

  • distance – a positive numeric value representing the maximum distance in kilometers between instrument1 and instrument2
