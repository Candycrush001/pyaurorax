aurorax.ephemeris module¶

class aurorax.ephemeris.EphemerisRecord(program=None, platform=None, instrument_type=None, epoch=None, geo_lat=None, geo_lon=None, gsm_lat=None, gsm_lon=None, nbtrace_lat=None, nbtrace_lon=None, sbtrace_lat=None, sbtrace_lon=None, metadata=None, ephemeris_source_identifier=None, location_geo=None, location_gsm=None, nbtrace=None, sbtrace=None)¶

Bases: object

Represents an AuroraX ephemeris record.

Variables

  • ephemeris_source_identifier – ID of the instrument’s Aurorax ephemeris source

  • program – the instrument’s program

  • platform – the instrument’s platform

  • instrument_type – the instrument’s type

  • epoch – datetime object representing the record’s time of occurrence

  • geo_lat – the instrument’s geographic latitude. Use with geo_lon, or use location_geo dictionary

  • geo_lon – the instrument’s geographic longitude. Use with geo_lat, or use location_geo dictionary

  • gsm_lat – the instrument’s GSM latitude. Use with gsm_lon, or use location_gsm dictionary

  • gsm_lon – the instrument’s GSM longitude. Use with gsm_lat, or use location_gsm dictionary

  • nbtrace_lat – the instrument’s north B trace geographic latitude. Use with nbtrace_lon, or use nbtrace dictionary

  • nbtrace_lon – the instrument’s north B trace geographic longitude. Use with nbtrace_lat, or use nbtrace dictionary

  • sbtrace_lat – the instrument’s south B trace geographic latitude. Use with sbtrace_lon, or use sbtrace dictionary

  • sbtrace_lon – the instrument’s south B trace geographic longitude. Use with sbtrace_lat, or use sbtrace dictionary

  • location_geo – the instrument’s geographic coordinates in a dictionary in the form {“lat: <LATITUDE>, “lon”: <LONGITUDE>}

  • location_gsm – the instrument’s GSM coordinates in a dictionary in the form {“lat: <LATITUDE>, “lon”: <LONGITUDE>}

  • nbtrace – the instrument’s north B trace geographic coordinates in a dictionary in the form {“lat: <LATITUDE>, “lon”: <LONGITUDE>}

  • sbtrace – the instrument’s south B trace coordinates in a dictionary in the form {“lat: <LATITUDE>, “lon”: <LONGITUDE>}

  • metadata – the instrument’s metadata dictionary

  • record – the ephemeris record’s fields in AuroraX dictionary format

to_dictionary()¶

Returns a copy of the EphemerisRecord as a dictionary in the AuroraX record format.

class aurorax.ephemeris.EphemerisSearch(programs=None, platforms=None, instrument_types=None, metadata_filters=None, start=None, end=None)¶

Bases: object

A class for searching ephemeris records.

Variables

  • start – a datetime object representing the search window’s start

  • end – a datetime object representing the seach window’s end

  • programs – a list of strings representing the programs to search

  • platforms – a list of strings representing the platforms to search

  • instrument_types – a list of strings representing the types of instrument to search

  • metadata_filters – a list of dictionaries representing metadata filters for the ground instrument in the form {“key”: <key>, “operator”: <operator>, “value”: <value>}. E.g. {“key”: “temperature”, “operator”: “=”, “value”: 25.0}

execute()¶

Executes the search with the given parameters.

Returns

a list of EphemerisRecord objects matching the search criteria, or an empty list if the search returned no records, or -1 if an error occurred

to_dictionary()¶

Returns a copy of the search parameters as a dictionary in the AuroraX ephemeris search format.

class aurorax.ephemeris.EphemerisUpload(api_key=None)¶

Bases: object

A class for uploading ephemeris records.

Variables

  • ephemeris_sources – a list of dictionary objetcs, each representing an AuroraX ephemeris source. Identical to the result of get_ephemeris_sources()

  • record_list – a list of EphemerisRecord objects to be uploaded

  • api_key – the user’s unique API key used for uploading ephemeris records to AuroraX. See the API for instructions on generating an API key

add_record(record)¶

Add an EphemerisRecord object to be uploaded.

Parameters

record – an EphemerisRecord object

Raise

ValueError if the record is not an EphemerisRecord object

clear()¶

Clears all records from record_list.

execute()¶

Sends all EphemerisRecord objects in the record_list to the API. A valid api_key variable must be defined.

The record_list will NOT be cleared after execution of this method. Use the clear() method to clear the record_list.

Returns

1 if all records were successfully uploaded, else returns the HTTP response returned on error

aurorax.ephemeris.get_ephemeris_sources(program=None, platform=None, instrument_type=None, source_type=None, owner=None, format='basic_info')¶

Returns a list of dictionaries representing all ephemeris sources.

Parameters

  • program – program name to filter sources by, optional

  • platform – platform name to filter sources by, optional

  • instrument_type – instrument type to filter sources by, optional

  • source_type – source type to filter sources by (e.g. “heo”), optional

  • owner – owner ID to filter sources by, optional

  • format – the format of the ephemeris source returned Available values: “identifier_only”, “basic_info”, “full_record”. Defaults to “basic_info”.

Returns

a dictionary of all ephemeris sources.
