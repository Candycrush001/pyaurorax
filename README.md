## Requirements
- Python 3.6+

## Installation
<pre>$ pip install pyaurorax</pre>

## Usage
<pre>import aurorax</pre>

## Development
<pre>$ git clone https://github.com/ucalgary-aurora/pyaurorax.git
$ cd pyaurorax
$ make install</pre>

## Testing
<pre>$ make test</pre>

## Additional Testing for Development Environments
<pre>$ make test-dev</pre>
